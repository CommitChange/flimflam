
# validated-form

```js
import {form, field, init as initForm} from 'ff-core/validated-form'

```

default validators: 

| key | val |
| --- | --- |
| email | boolean |
| required | boolean |
| currency | boolean |
| format | regex |
| isNumber | boolean |
| max | integer |
| min | integer |
| equalTo | string | 
| maxLength | integer |
| minLength | integer |
| lengthEquals | integer |
| includedIn | array |
